# Plug and Play

## Overview
Plug and Play is a full-stack e-commerce website project designed to provide users with a smooth and responsive online shopping experience. The project focuses on product browsing, user interaction, authentication, and organized product presentation in a modern web environment.

## Features
- Product listing and browsing
- Product detail viewing
- User authentication interface
- Responsive user interface
- Structured product organization

## Tools and Technologies Used
- HTML
- CSS
- JavaScript
- React
- Node.js
- Express.js
- MongoDB
- Tailwind CSS

## My Role
I worked on the development and structure of the website, including the front-end interface, product presentation, and integration of key website components.
